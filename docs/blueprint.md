# **App Name**: SpaceWise

## Core Features:

- Location Mapping: Integrate with Google Maps to display locations and their overall occupancy. The map should be opened directly on the Netherlands.
- Room and Bed Management: Define rooms within locations and beds within rooms (predefined, not editable in-app).
- Occupancy Tracking: Track occupancy status (available, occupied, reserved) for each bed, with the ability to add/remove occupants and select the period of occupancy.
- Availability Calendar: A calendar that displays availability and reservations per location.
- Booking Tool: Use a generative AI tool to suggest the most adequate bed depending on space available and dates.
- Authentication: OIDC Authentication flow with Google Account Sign-In
- Firebase Integration: Store data in Firebase database
- Location List: Display an expandable list with all locations and their details.
- Language Switch: Button to switch the language to English and the default should be Dutch.

## Style Guidelines:

- Primary color: White (#FFFFFF).
- Background color: Light gray (#F0F0F0), providing a neutral backdrop for clear data presentation.
- Accent color: Red-orange (#FF5733) used to highlight actionable items.
- Font pairing: 'Inter' sans-serif for body text and 'Space Grotesk' for headings, for clear hierarchy.
- Simple, modern icons to represent locations, rooms, and beds, ensuring ease of navigation.
- Clean, structured layouts with ample white space to avoid visual clutter, optimizing for data-heavy displays.
- Smooth transitions and subtle animations on data updates to enhance user experience.
